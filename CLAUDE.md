# image_creator - Ray.so Image Card Generator

Next.js + Tailwind CSS + Jotai + Radix UI + Shiki

<directory>
api/ - API routes for backend logic (e.g. image proxy, snippets)
app/ - Next.js App Router root, UI pages and layouts
components/ - Reusable UI components (Editor, Toolbar, etc.)
public/ - Static assets (fonts, icons, themes)
utils/ - Shared utility functions (styling, parsing, store)
</directory>

<config>
package.json - Dependency and script definitions
tailwind.config.ts - Tailwind CSS theme configuration
next.config.js - Next.js configuration and rewrites
tsconfig.json - TypeScript configuration
</config>

法则: 极简·稳定·导航·版本精确
